Usage

Html

<div class="modal">
</div>

Properties

Property Attribute Description Type Default
animated animated If true, the modal will animate. boolean true
backdropDismiss backdrop-dismiss If true, the modal will be dismissed when the backdrop is clicked. boolean true
closeButton close-button If true, the modal will have a close button on large screens. boolean true
component (required) component The component to display inside of the modal. Function \| HTMLElement \| string undefined
componentProps – The data to pass to the modal component. { [key: string]: any; } undefined
cssClass css-class Additional classes to apply for custom CSS. If multiple classes are provided they should be separated by spaces. string \| string[] undefined
enterAnimation – Animation to use when the modal is presented. (baseEl: any, opts?: any) => Animation undefined
indicator indicator If true, the modal will have a drag indicator. boolean true
keyboardClose keyboard-close If true, the keyboard will be automatically dismissed when the overlay is presented. boolean true
leaveAnimation – Animation to use when the modal is dismissed. (baseEl: any, opts?: any) => Animation undefined
presentingElement – The element that presented the modal. This is used for card presentation effects and for stacking multiple modals on top of each other HTMLElement document.querySelector('bkkr-root') \|\| document.body
showBackdrop show-backdrop If true, a backdrop will be displayed behind the modal. boolean true
swipeToClose swipe-to-close If true, the modal can be swiped to dismiss. boolean true

Events

Event Description Type
bkkrModalDidDismiss Emitted after the modal has dismissed. CustomEvent<OverlayEventDetail<any>>
bkkrModalDidPresent Emitted after the modal has presented. CustomEvent<void>
bkkrModalWillDismiss Emitted before the modal has dismissed. CustomEvent<OverlayEventDetail<any>>
bkkrModalWillPresent Emitted before the modal has presented. CustomEvent<void>

Methods

dismiss(data?: any, role?: string) => Promise<boolean>

Dismiss the modal overlay after it has been presented.

Returns

Type: Promise<boolean>

onDidDismiss<T = any>() => Promise<OverlayEventDetail<T>>

Returns a promise that resolves when the modal did dismiss.

Returns

Type: Promise<OverlayEventDetail<T>>

onWillDismiss<T = any>() => Promise<OverlayEventDetail<T>>

Returns a promise that resolves when the modal will dismiss.

Returns

Type: Promise<OverlayEventDetail<T>>

present() => Promise<void>

Present the modal overlay after it has been created.

Returns

Type: Promise<void>
