Usage

Angular / javascript

<bkkr-item>
    <bkkr-label>Gender</bkkr-label>
    <bkkr-select placeholder="Select One">
      <bkkr-select-option value="f">Female</bkkr-select-option>
      <bkkr-select-option value="m">Male</bkkr-select-option>
    </bkkr-select>
  </bkkr-item>

Properties

Property Attribute Description Type Default
cancelText cancel-text The text to display on the cancel button. string 'Cancel'
compareWith compare-with A property name or function used to compare object values ((currentValue: any, compareValue: any) => boolean) | string undefined
disabled disabled If true, the user cannot interact with the input. boolean false
interface interface The interface the select should use: drawer, popover or alert. "alert" | "drawer" | "popover" 'popover'
interfaceOptions interface-options Any additional options that the alert, drawer or popover interface can take. See the bkkr-alert docs and the bkkr-popover docs for the create options for each interface. Note: interfaceOptions will not override inputs or buttons with the alert interface. any {}
multiple multiple If true, the select can accept multiple values. boolean false
name name The name of the control, which is submitted with the form data. string this.inputId
okText ok-text The text to display on the ok button. string 'OK'
placeholder placeholder The text to display when the select is empty. string undefined
search search If true, a searchbar appears. This is not supported on ‘alert’ interface. boolean false
selectedText selected-text The text to display instead of the selected option’s value. string undefined
toggleIcon toggle-icon If true, a toggle icon will display. boolean true
value value the value of the select. any undefined

Events

Event Description Type
bkkrBlur Emitted when the select loses focus. CustomEvent<void>
bkkrCancel Emitted when the selection is cancelled. CustomEvent<void>
bkkrChange Emitted when the value has changed. CustomEvent<SelectChangeEventDetail<any>>
bkkrFocus Emitted when the select has focus. CustomEvent<void>
bkkrOpen Emitted when the selection is opened. CustomEvent<void>
bkkrSearch Emitted when the user types in a searchbar. CustomEvent<SelectChangeEventDetail<any>>

Methods

open(event?: UIEvent) => Promise<any>

Open the select overlay. The overlay is either an alert, action sheet, or popover, depending on the interface property on the bkkr-select.

Returns

Type: Promise<any>

Shadow Parts

Part Description
"icon" The select icon container.
"placeholder" The text displayed in the select when there is no value.
"text" The displayed value of the select.

CSS Custom Properties

Name Description
--padding-bottom Bottom padding of the select
--padding-end Right padding if direction is left-to-right, and left padding if direction is right-to-left of the select
--padding-start Left padding if direction is left-to-right, and right padding if direction is right-to-left of the select
--padding-top Top padding of the select
--placeholder-color Color of the select placeholder text
--placeholder-opacity Opacity of the select placeholder text
