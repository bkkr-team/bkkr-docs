Events

Event Description Type
bkkrBlur Emitted when the input loses focus. CustomEvent<void>
bkkrCancel Emitted when the cancel button is clicked. CustomEvent<void>
bkkrChange Emitted when the value has changed. CustomEvent<SearchbarChangeEventDetail>
bkkrClear Emitted when the clear input button is clicked. CustomEvent<void>
bkkrFocus Emitted when the input has focus. CustomEvent<void>
bkkrInput Emitted when a keyboard input occurred. CustomEvent<KeyboardEvent>
