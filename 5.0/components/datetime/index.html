[datetime]

Usage

Angular / html

<!-- Default -->
<bkkr-datetime></bkkr-datetime>

<!-- Datetime with value -->
<bkkr-datetime value="1990-02-19"></bkkr-datetime>

<!-- Datetime with placeholder -->
<bkkr-datetime placeholder="Enter datetime"></bkkr-datetime>

Methods

getdatetimeElement() => Promise<HTMLInputElement>

Returns the native <datetime> element used under the hood.

Returns

Type: Promise<HTMLInputElement>

open(event?: UIEvent) => Promise<any>

Open the select overlay. The overlay is either an alert, action sheet, or popover, depending on the interface property on the bkkr-select.

Returns

Type: Promise<any>

setFocus() => Promise<void>

Sets focus on the native datetime in bkkr-datetime. Use this method instead of the global datetime.focus().

Returns

Type: Promise<void>
