[Header]

Usage

Angular / html

<!-- Default -->
<bkkr-header>
  <bkkr-toolbar>
    <bkkr-buttons slot="start">
      <bkkr-back-button></bkkr-back-button>
    </bkkr-buttons>
    <bkkr-title>My Navigation Bar</bkkr-title>
  </bkkr-toolbar>

  <bkkr-toolbar>
    <bkkr-title>Subheader</bkkr-title>
  </bkkr-toolbar>
</bkkr-header>

<!-- Header without a border -->
<bkkr-header class="border-0">
  <bkkr-toolbar>
    <bkkr-title>Header - No Border</bkkr-title>
  </bkkr-toolbar>
</bkkr-header>

<bkkr-content></bkkr-content>

Properties

Property Attribute Description Type Default
color color The color to use from your application’s color palette. Default options are: "primary", "secondary", "tertiary", "success", "warning", "danger", "light", "medium", and "dark". For more information on colors, see theming. string undefined

Name Description
