Usage

Angular

<bkkr-list>
  <bkkr-item *ngFor="let item of items">
    <bkkr-thumbnail slot="start">
      <bkkr-img [src]="item.src"></bkkr-img>
    </bkkr-thumbnail>
    <bkkr-label>{{item.text}}</bkkr-label>
  </bkkr-item>
</bkkr-list>

Html

<bkkr-list>
  <bkkr-item>
    <bkkr-thumbnail slot="start">
      <bkkr-img src="https://cdn.pixabay.com/photo/2017/08/30/01/05/milky-way-2695569__340.jpg"></bkkr-img>
    </bkkr-thumbnail>
    <bkkr-label>Milky way</bkkr-label>
  </bkkr-item>

  <bkkr-item>
    <bkkr-thumbnail slot="start">
      <bkkr-img src="https://cdn.pixabay.com/photo/2017/07/03/20/17/abstract-2468874__340.jpg"></bkkr-img>
    </bkkr-thumbnail>
    <bkkr-label>Abstract colors</bkkr-label>
  </bkkr-item>
</bkkr-list>

Properties

Property Attribute Description Type Default
alt alt This attribute defines the alternative text describing the image. Users will see this text displayed if the image URL is wrong, the image is not in one of the supported formats, or if the image is not yet downloaded. string undefined
src src The image URL. This attribute is mandatory for the <img> element. string undefined

Events

Event Description Type
bkkrError Emitted when the img fails to load CustomEvent<void>
bkkrImgDidLoad Emitted when the image has finished loading CustomEvent<void>
bkkrImgWillLoad Emitted when the img src has been set CustomEvent<void>
