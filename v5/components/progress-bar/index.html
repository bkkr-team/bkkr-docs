Usage

Angular

<!-- Default Progressbar -->
<bkkr-progress-bar></bkkr-progress-bar>

<!-- Default Progressbar with 50 percent -->
<bkkr-progress-bar [value]="0.5"></bkkr-progress-bar>

<!-- Colorize Progressbar -->
<bkkr-progress-bar color="primary" [value]="0.5"></bkkr-progress-bar>
<bkkr-progress-bar color="secondary" [value]="0.5"></bkkr-progress-bar>

<!-- Other types -->
<bkkr-progress-bar [value]="0.25" [buffer]="0.5"></bkkr-progress-bar>
<bkkr-progress-bar type="indeterminate"></bkkr-progress-bar>
<bkkr-progress-bar type="indeterminate" reversed="true"></bkkr-progress-bar>

Javascript

<!-- Default Progressbar -->
<bkkr-progress-bar></bkkr-progress-bar>

<!-- Default Progressbar with 50 percent -->
<bkkr-progress-bar value="0.5"></bkkr-progress-bar>

<!-- Colorize Progressbar -->
<bkkr-progress-bar color="primary" value="0.5"></bkkr-progress-bar>
<bkkr-progress-bar color="secondary" value="0.5"></bkkr-progress-bar>

<!-- Other types -->
<bkkr-progress-bar value="0.25" buffer="0.5"></bkkr-progress-bar>
<bkkr-progress-bar type="indeterminate"></bkkr-progress-bar>
<bkkr-progress-bar type="indeterminate" reversed="true"></bkkr-progress-bar>

Properties

Property Attribute Description Type Default
buffer buffer If the buffer and value are smaller than 1, the buffer circles will show. The buffer should be between [0, 1]. number 1
color color The color to use from your application’s color palette. Default options are: "primary", "secondary", "tertiary", "success", "warning", "danger", "light", "medium", and "dark". For more information on colors, see theming. string | undefined undefined
reversed reversed If true, reverse the progress bar direction. boolean false
type type The state of the progress bar, based on if the time the process takes is known or not. Default options are: "determinate" (no animation), "indeterminate" (animate from left to right). "determinate" | "indeterminate" 'determinate'
value value The value determines how much of the active bar should display when the type is "determinate". The value should be between [0, 1]. number 0

CSS Custom Properties

Name Description
--background Same as –buffer-background when using a determinate progress bar, otherwise it styles the background of the bkkr-progress-bar itself.
--buffer-background Color of the buffer bar
--progress-background Color of the progress bar
