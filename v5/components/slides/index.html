Usage

Angular / javascript

<bkkr-slides pager="true">
    <bkkr-slide>
      <h1>Slide 1</h1>
    </bkkr-slide>

    <bkkr-slide>
      <h1>Slide 2</h1>
    </bkkr-slide>

    <bkkr-slide>
      <h1>Slide 3</h1>
    </bkkr-slide>
</bkkr-slides>

Properties

Property Attribute Description Type Default
options – Options to pass to the swiper instance. See http://idangero.us/swiper/api/ for valid options SwiperOptions {}
pager pager If true, show the pagination. boolean false
scrollbar scrollbar If true, show the scrollbar. boolean false

Events

Event Description Type
bkkrSlideDidChange Emitted after the active slide has changed. CustomEvent<void>
bkkrSlideDoubleTap Emitted when the user double taps on the slide’s container. CustomEvent<void>
bkkrSlideDrag Emitted when the slider is actively being moved. CustomEvent<void>
bkkrSlideNextEnd Emitted when the next slide has ended. CustomEvent<void>
bkkrSlideNextStart Emitted when the next slide has started. CustomEvent<void>
bkkrSlidePrevEnd Emitted when the previous slide has ended. CustomEvent<void>
bkkrSlidePrevStart Emitted when the previous slide has started. CustomEvent<void>
bkkrSlideReachEnd Emitted when the slider is at the last slide. CustomEvent<void>
bkkrSlideReachStart Emitted when the slider is at its initial position. CustomEvent<void>
bkkrSlidesDidLoad Emitted after Swiper initialization CustomEvent<void>
bkkrSlideTap Emitted when the user taps/clicks on the slide’s container. CustomEvent<void>
bkkrSlideTouchEnd Emitted when the user releases the touch. CustomEvent<void>
bkkrSlideTouchStart Emitted when the user first touches the slider. CustomEvent<void>
bkkrSlideTransitionEnd Emitted when the slide transition has ended. CustomEvent<void>
bkkrSlideTransitionStart Emitted when the slide transition has started. CustomEvent<void>
bkkrSlideWillChange Emitted before the active slide has changed. CustomEvent<void>
