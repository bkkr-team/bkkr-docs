Events

Event Description Type
bkkrBlur Emitted when the input loses focus. CustomEvent<FocusEvent>
bkkrChange Emitted when the value has changed. CustomEvent<InputChangeEventDetail>
bkkrFocus Emitted when the input has focus. CustomEvent<FocusEvent>
bkkrInput Emitted when a keyboard input occurred. CustomEvent<KeyboardEvent>
