Properties​

color​

DescriptionThe color to use for the background of the item.
Attributecolor
Typestring ｜ undefined
Defaultundefined

icon​

DescriptionA combination of both name and src. If a src url is detected
it will set the src property. Otherwise it assumes it's a built-in named
SVG and set the name property.
Attributeicon
Typeany
Defaultundefined

lazy​

DescriptionIf enabled, bkkr-icon will be loaded lazily when it's visible in the viewport.
Default, false.
Attributelazy
Typeboolean
Defaultfalse

name​

DescriptionThe name of icons.
See all icons on Fontawesome.
Attributename
Typestring ｜ undefined
Defaultundefined

sanitize​

DescriptionWhen set to false, SVG content that is HTTP fetched will not be checked
if the response SVG content has any <script> elements, or any attributes
that start with on, such as onclick.
Attributesanitize
Typeboolean
Defaulttrue

size​

DescriptionThe icon size.
Attributesize
Type"default" ｜ "large" ｜ "small" ｜ undefined
Defaultundefined

src​

DescriptionSpecifies the exact src of an SVG file to use.
Attributesrc
Typestring ｜ undefined
Defaultundefined

type​

DescriptionThe icon size.
Attributetype
Type"brands" ｜ "regular" ｜ "solid"
Default'solid'
